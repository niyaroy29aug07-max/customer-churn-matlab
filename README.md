# Customer Churn Prediction (MATLAB)

## 📌 Overview
This project predicts whether a customer will churn using a Decision Tree model in MATLAB.

## ⚙️ Features
- Data cleaning
- Feature selection
- Train-test split
- Decision Tree model
- Evaluation (Accuracy, Precision, Recall)

## 📊 Dataset
Telco Customer Churn Dataset

## 🚀 How to Run
1. Download this repository
2. Open in MATLAB
3. Run `churn_project.m`

## 📈 Output
- Accuracy score
- Confusion matrix
- Decision tree visualization
